### Examples

Two worked FSH examples, both rendered as part of this IG.

| | |
| :--- | :--- |
| [Bundle: single-visit example](Bundle-ExampleAncBundleSingleVisit.md) | The canonical first-ANC-visit payload: 1 Patient, 1 Organization, 1 finished Encounter, 1 planned Encounter, 4 Observations. |
| [Bundle: multi-visit example](Bundle-ExampleAncBundleMultiVisit.md) | Same patient, same facility,**three**finished ANC visits + 1 planned. Use this as the reference for reporting repeat visits in one Bundle. |

Both use the same [example patient](Patient-EnablePatientExample.md) (MRN `ANC-000718642`) and the same [example facility](Organization-ExampleFacility.md) (DHIS2 code `1057888`, "Felege Mels Health Center").
